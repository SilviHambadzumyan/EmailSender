# EmailSender

PRD text

Who?

Our target audience is businesses and individuals who need to send emails and follow ups to their potential clients/partners. 

Why?

With the help of our platform, our clients will be able to send emails and follow ups and easily keep track of them without any manual work. 

When?

We need to finish the project by the end of the current semester. 

What?

A software platform that will ease the life of our potential customers for sending emails. 

How?
We will use Python, JavaScript, CSS and HTML.
